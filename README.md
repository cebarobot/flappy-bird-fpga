# Flappy Bird on FPGA

## Requirements
* For FPGA: [TANG Nano 9K](https://wiki.sipeed.com/hardware/en/tang/Tang-Nano-9K/Nano-9K.html) with 5inch LCD screen.
* For Simulation: Verilator with SDL.
